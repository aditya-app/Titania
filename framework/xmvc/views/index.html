Thank You For Using Titania Framework!

Titania is a ruby web framework, (that i hope are) simple, easy to use, and fast. Developed by Aditya "Avaga" Purwa of ION.

This version you're using is unstable preview of Titania, on its development preview. There is no native database support, no plug-ins, libraries, etc. The only thing you can get from this version is an unstable XMVC working set.

We introduce you to the new world of MVC, which is the XMVC (Extended Model View Controller). XMVC is the main design pattern used in developing web application using this framework.

Please note that Titania is currently on its unstable preview version. Future changes may occur frequently during this development period. The date of stable release is still unknown.
